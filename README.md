# Zig wrapper

Wrapper for Zig inspired by [Maven Wrapper](https://maven.apache.org/wrapper/)

## Usage

Run `zig-wrapper.sh` in the directory containing `build.zig` for your project after which `./zigw` can be used for all `zig` commands.
